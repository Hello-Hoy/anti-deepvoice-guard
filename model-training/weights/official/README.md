# Official AASIST Weights (ASVspoof 2019 LA)

clovaai/aasist 공식 사전학습 가중치.

- Source: https://github.com/clovaai/aasist
- Dataset: ASVspoof 2019 LA (Logical Access)
- EER: 0.83%, min t-DCF: 0.0275
- Loss: CrossEntropyLoss (softmax inference)

## Download
`download_official_weights.py` 스크립트를 실행하거나, 수동으로 다운로드:

```bash
cd model-training
python download_official_weights.py
```
