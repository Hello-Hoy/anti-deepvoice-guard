# KTM Mobile Reference Button Kit

This folder separates the `ktm_mobile/` reference screenshots into button-level image assets.

## Structure

- `home-header/`: top header buttons from the main page, such as search, profile, and menu.
- `home-tabs/`: horizontal category tabs from the main page.
- `top-icons/`: menu overlay top icon buttons, such as home, logout, settings, and close.
- `quick-actions/`: four illustrated quick action buttons.
- `side-menu/`: left-side category menu buttons, including active and inactive states.
- `right-menu/`: right-side list/menu row buttons.

## Style Guide For Similar App Assets

Use these references as layout and state examples, not as brand copies.

- Visual style: clean Korean mobile carrier app UI, white background, generous spacing, black text, gray inactive state, black active state.
- Navigation pattern: fixed top utility icons, four quick action buttons, two-column drawer menu with a left category rail and right menu list.
- Icon style: simple line icons for utility actions; small colorful flat illustrations for quick actions.
- State treatment: active side category uses black background with white text; inactive side category uses light gray background with dark gray text.
- Button separation: keep each button exported as its own PNG so it can be used independently in mockups or generation prompts.

## Suggested Generation Prompt Template

Create a modern Korean telecom mobile app UI asset inspired by the provided reference structure, without copying any brand logo or exact trademark. White background, clean spacing, bold Korean typography, simple black line icons, selective colorful flat illustrations, minimal commercial app style. Generate only the requested button asset as an isolated PNG-like UI element, centered, with no extra screen frame, no watermark, and no unrelated decoration.

Replace `requested button asset` with one of:

- top utility icon: home, logout, settings, close, search, profile, menu
- quick action: plan, application lookup, FAQ, SIM replacement
- side menu category: join, products, benefits, bundle, customer center, my page
- right menu row: plan, value added service, extra service, phone, discount benefits, bundle plan, customer notice, member info
