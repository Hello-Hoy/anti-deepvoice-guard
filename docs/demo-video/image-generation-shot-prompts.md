# ChatGPT Image Shot Prompts

Use these prompts for ChatGPT image generation. They follow the reference prompt style from the Notion examples: each prompt states the asset purpose, visual composition, exact copy, mood, commercial usability, and aspect ratio in one production-ready brief.

Important generation rule:

```text
이미지 안의 한글과 영어 문구는 아래에 적힌 문구 그대로 정확히 표시해 주세요. 글자를 임의로 바꾸거나 깨진 문자처럼 만들지 말고, 발표 화면에서 읽히는 선명한 UI 텍스트로 구성해 주세요.
```

## Common Screen System

Use this common system for every phone-screen image so the final video feels like one product demo, not separate generated assets.

```text
공통 화면 규칙: 모든 스마트폰 화면은 물리적 휴대폰 프레임, 책상, 식물, 카메라 원근감 없이 직접 캡처한 Android 스크린샷처럼 만든다. 모든 결과물은 같은 캔버스 크기와 같은 화면비를 사용한다. 기준은 첨부한 실제 휴대폰 캡처처럼 좁고 긴 세로 화면이며, 권장 출력은 896x2048px 또는 그에 준하는 동일 비율이다. 서로 다른 샷 사이에서 화면 폭, 세로 길이, 상태창 높이, 하단 홈 인디케이터 위치가 달라지면 안 된다.

상태창 고정 규칙: 모든 이미지의 상태창은 완전히 동일해야 한다. 왼쪽 상단에는 "9:41"만 표시한다. 오른쪽 상단에는 같은 모양의 흰색/검은색 신호 막대 아이콘, Wi-Fi 아이콘, 배터리 아이콘, "100%" 텍스트만 표시한다. 아이콘 개수, 아이콘 순서, 배터리 모양, 글자 크기, 좌우 여백을 모든 이미지에서 동일하게 유지한다. 통신사명 SKT, KT, LTE, 5G, 무음 아이콘, 날씨 위젯, 추가 알림, 실제 브랜드 로고는 넣지 않는다.

푸쉬알림 잠금화면 규칙: 배경은 두 푸쉬알림 이미지에서 완전히 같은 짙은 회색 추상 월페이퍼를 사용한다. 중앙 큰 시간 "9:41", 날짜 "5월 16일 금요일", 잠금 아이콘, 하단 전화 아이콘, 카메라 아이콘, 홈 인디케이터의 위치와 크기를 두 이미지에서 동일하게 유지한다. 알림 카드는 화면 폭의 약 84~86%를 차지하는 밝은 회색 둥근 사각형이며, 카드의 x/y 위치, 폭, 높이, 모서리 반경, 그림자를 두 이미지에서 동일하게 맞춘다. 왼쪽 앱 아이콘은 두 이미지 모두 같은 빨간 방패형 느낌표 아이콘을 사용한다. 아이콘 색을 주황색으로 바꾸지 않는다. 앱 이름 "TrueVoice Guard"는 두 이미지 모두 같은 보라색, 오른쪽 시각 "지금"은 같은 짙은 회색, 본문은 두 이미지 모두 검은색으로 표시한다.

앱 분석화면 규칙: 흰색 배경, 보라색 TrueVoice Guard 로고/브랜드, 보라색 음성 파형, 둥근 카드형 UI, 하단 내비게이션 4개 탭을 모든 분석 화면에서 동일하게 유지한다. 하단 탭은 왼쪽부터 "실시간 분석", "기록", "가족 보호", "설정"이며 아이콘은 두 분석 이미지에서 같은 모양이어야 한다. 선택된 첫 번째 탭의 아이콘과 텍스트는 두 이미지 모두 보라색, 나머지 탭 아이콘과 텍스트는 모두 같은 회색이다. DANGER 화면은 위험 수치와 최종 카드만 빨간색, WARNING 화면은 피싱 수치와 최종 카드만 주황색으로 바꾸고, 상태창/상단 로고/하단 내비게이션/카드 위치/아이콘 모양은 바꾸지 않는다.
```

## Consistency Workflow For ChatGPT Image2

For the four key phone assets, do not generate all four from scratch. Use a base-and-edit workflow.

```text
1. 먼저 Shot 06 기준 잠금화면을 생성한다.
2. Shot 10은 Shot 06 결과 이미지를 첨부한 뒤 "텍스트만 교체하고 나머지는 픽셀 단위로 유지"하는 편집 프롬프트로 만든다.
3. 먼저 Shot 05 기준 분석화면을 생성한다.
4. Shot 09는 Shot 05 결과 이미지를 첨부한 뒤 "수치, 문구, 위험 색상만 변경하고 상태창/하단 내비게이션/레이아웃은 픽셀 단위로 유지"하는 편집 프롬프트로 만든다.
```

If ChatGPT Image2 allows only one image at a time, paste the relevant edit prompt and attach the base image. If it changes the icon set, status bar, aspect ratio, or nav icons, reject the result and regenerate with the "Lock Invariants" text below.

### Lock Invariants

```text
잠금 조건: 첨부한 기준 이미지의 캔버스 크기, 화면비, 상태창, 잠금화면 배경, 알림 카드 위치, 앱 아이콘, 하단 전화/카메라 아이콘, 홈 인디케이터를 변경하지 마세요. 다른 디자인으로 다시 그리지 말고, 기준 이미지 위에서 지정된 텍스트만 바꾸는 이미지 편집처럼 처리하세요. 아이콘 색상, 아이콘 모양, 글자 색, 카드 크기, 여백, 월페이퍼 곡선은 기준 이미지와 동일해야 합니다.
```
